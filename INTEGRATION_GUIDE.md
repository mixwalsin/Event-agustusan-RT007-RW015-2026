Update Database Credentials\n\n**api/config.php:**\n\n```php\ndefine('DB_HOST', 'your-cPanel-database-host');\ndefine('DB_USER', 'your_cpanel_user_dbname');\ndefine('DB_PASS', 'your_secure_password');\ndefine('DB_NAME', 'agustusan_rt007_2026');\n```\n\n**Get info dari cPanel:**\n- Login ke cPanel\n- Cari \"MySQL Databases\" atau \"MariaDB\"\n- Copy hostname, username, database name\n\n### 2. Update API Base URL\n\n**js/api.js:**\n\n```javascript\n// Production config\nconst API_BASE_URL = \"https://yourdomain.com/api\";\n\n// Atau dengan subfolder\nconst API_BASE_URL = \"https://yourdomain.com/path/to/Event-agustusan-RT007-RW015-2026/api\";\n```\n\n### 3. Secure Configuration\n\n```php\n// api/config.php\n\n// Disable debug mode\ndefine('DEBUG_MODE', false);\n\n// Change JWT secret\ndefine('JWT_SECRET', 'generate-random-string-here');\n\n// Restrict CORS untuk production (optional)\nheader('Access-Control-Allow-Origin: https://yourdomain.com');\nheader('Access-Control-Allow-Origin: https://username.github.io');\n```\n\n### 4. Update Admin Password\n\n1. **Via phpMyAdmin:**\n   - Select database `agustusan_rt007_2026`\n   - Select table `admin`\n   - Edit row dengan username `admin`\n   - Update `password` field dengan hash baru\n\n2. **Generate hash (PHP):**\n   ```php\n   <?php\n   $password = 'new-secure-password';\n   echo password_hash($password, PASSWORD_BCRYPT);\n   ?>\n   ```\n\n3. **Or update via script:**\n   ```bash\n   php -r \"echo password_hash('new-password', PASSWORD_BCRYPT);\"\n   ```\n\n---\n\n## 📦 Deployment Strategy\n\n### Option 1: Frontend di GitHub Pages + Backend di cPanel\n\n**Frontend Setup:**\n\n1. // Should match phpMyAdmin\necho DB_USER; // Should be valid\necho DB_NAME; // Should exist\n```\n\n### Issue: 404 on API Endpoints\n\n- Check file names (.php files exist)\n- Check file permissions (644)\n- Check URL structure di `js/api.js`\n\n### Issue: Slow API Response\n\n- Check database indexes\n- Limit query results\n- Enable caching\n- Monitor database size\n\n---\n\n## 📚 Resources\n\n- [README_BACKEND.md](README_BACKEND.md) - Backend setup detail\n- [QUICK_START.md](QUICK_START.md) - Quick reference\n- [API Endpoints Documentation](#api-documentation)\n\n---\n\n**Made with ❤️ for SEMARAK AGUSTUS 2026**\n"
